 # Part2 - RecipeApp 
Description:
The Recipe App is a console-based application designed to help users manage their recipes. Users can enter new recipes, view existing recipes, scale recipes, reset quantities, and clear recipes. Additionally, the app calculates the total calories of each recipe and notifies the user if a recipe exceeds 300 calories.
## How to Use
After running the application, you'll see the main menu displayed in the console.
The menu will list various options for interacting with the Recipe App.
Select an Option:

Use the keyboard to select an option from the menu by entering the corresponding number and pressing Enter.
Here are the available options:
Enter a new recipe
View all recipes
Exit the application
Enter a New Recipe:

If you choose to enter a new recipe, follow the prompts provided by the application.
You'll be asked to enter the name of the recipe, the number of ingredients, details for each ingredient (name, quantity, unit, calories, food group), the number of steps, and details for each step.
View All Recipes:

If you select the option to view all recipes, the application will display a list of all existing recipes in alphabetical order by name.
You'll then be prompted to enter the name of the recipe you want to view.
The application will display the selected recipe along with its ingredients and steps.
Exit the Application:

To exit the application, select the option to exit from the main menu.
The application will display a goodbye message and close.
